# jupyterlab-icons-viewer

[![Page screenshot](screenshot.png)](https://jupyterlab-icons-viewer.surge.sh/)

## Development

- `npm install`
- `npm run dev`

## Deployment ([Surge](https://surge.sh/))

- `npm install -g surge` (if needed)
- `npm run build`
- `surge dist`
- Domain: https://jupyterlab-icons-viewer.surge.sh/
